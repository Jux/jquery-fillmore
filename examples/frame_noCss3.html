Frame Demo

This is a demo of the mode: 'frame' option, with the CSS3 option disabled.

Normal background-image

Doesn't shrink to fit.

background-size: contain;

Stretches the image.

$(…).fillmore({ …, mode: 'frame' });

Aah, just right.


<script src="../lib/jquery.js"></script>
<script src="../jquery.fillmore.js"></script>
<script>
  $.fillmore( {
    noCss3: true,  // disable CSS3 implementation, even if the browser supports it
    src: "pot-holder.jpg",
    mode: "frame"
  } );
</script>
