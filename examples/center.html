Centering Demo

This demo shows how to use jQuery Fillmore to specifify the desired center of a photo.

<script src="../lib/jquery-1.7.1.min.js"></script>
<script src="../jquery.fillmore.js"></script>
<script>
	$('.fillmored.center').fillmore( {
		src: "pot-holder.jpg"
		centerX: 75,  // a number between 1 and 100, starting from left
		centerY: 16   // ditto, starting from top
	} );
</script>
